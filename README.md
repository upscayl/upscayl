# ğŸ¤“ FAQ

  - **How does Upscayl work?**
    Â  - Upscayl uses AI models to enhance your images by guessing what the details could be. It uses Real-ESRGAN and Vulkan architecture to achieve this. \<a href="[https://github.com/upscayl/upscayl-ncnn](https://github.com/upscayl/upscayl-ncnn)" target="\_blank"\>Our backend\</a\> is fully open-source under the AGPLv3 license.
  - **I don't see a drastic change in my upscaled image. Why is that?**
    Â  - Upscayl can enhance low resolution images and images that are pixelated but it cannot de-blur or do focus adjustment on your image. If your image is out-of-focus or totally blurred, Upscayl is not the right tool for it. Please use images that are similar to the \<a href="COMPARISONS.MD" target="\_blank"\>examples we've given here.\</a\>
  - **Is there a CLI available?**
    Â  - The CLI tool is called \<a href="[https://github.com/upscayl/upscayl-ncnn](https://github.com/upscayl/upscayl-ncnn)" target="\_blank"\>upscayl-ncnn\</a\>.
  - **Do I need a GPU for this to work?**
    Â  - Yes, unfortunately. NCNN Vulkan requires a Vulkan-compatible GPU. Upscayl won't work with **most** iGPUs or CPUs. But hey, no harm in trying ;)
    Â  Â  - @Wyrdgirn has contributed a workaround for Windows and Linux in \<a href="[https://github.com/upscayl/upscayl/issues/390](https://github.com/upscayl/upscayl/issues/390)" target="\_blank"\>\#390\</a\>\! Nobody knows how to manipulate the macOS and Haiku frameworks...
  - **I stopped the magic Batch Upscayl and my images haven't been processed, compressed, or are in the wrong scale\!**
    Â  - When a model doesn't support an action, Upscayl will finish upscayling all the images first before post-processing them. What this means is that you should simply **wait** for the process to finish.
  - **How can I contribute?**
    Â  - You can report issues, fix code and add features by submitting PRs, or donate\! ğŸ˜Š
  - **What's the GPU ID for?**
    Â  - It is for selecting which GPU to use. The specific procedure is detailed in the \<a href="[https://github.com/upscayl/upscayl/wiki/Guide](https://github.com/upscayl/upscayl/wiki/Guide)" target="\_blank"\>Wiki\</a\>.
